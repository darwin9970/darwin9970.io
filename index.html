定义

浅拷贝：将数据中所有的数据引用下来，并指向同一个存放地址，拷贝的数据修改之后，会对原数据产生副作用。

深拷贝：将数据中所有数据拷贝下来，对拷贝之后的数据进行修改不会对原数据产生副作用。

浅拷贝

业务中，很多时候做的是浅拷贝，如果不影响业务逻辑，你可能不关心这些东西。

作用域中变量（标识符）的查找规则

首先声明一点，JavaScript是有编译过程的，不要惊讶，真的有！也就是说var name = 'iceman'这段代码，其实这是有两个动作的：

  • 编译器在当前作用域中声明一个变量name
  • 运行时引擎在作用域中查找该变量，找到了name变量并为其赋值

证明以上的说法：

在var name = 'iceman'的上一行输出name变量，并没有报错，输出undefined，说明输出的时候该变量已经存在了，只是没有赋值而已。

其实编译器是这样工作的，在代码执行之前从上到下的进行编译，当遇到某个用var声明的变量的时候，先检查在当前作用域下是否存在了该变量。如果存在，则忽略这个声明；如果不存在，则在当前作用域中声明该变量。

上面的这段简单的代码包含两种查找类型：输出变量的值的时候的查找类型是RHS，找到变量为其赋值的查找类型是LHS。

我猜各位同学一定可以猜到“L”和“R”的含义，这里的左侧和右侧指的是在赋值操作的左侧和右侧。也就是说，变量出现在赋值操作的左侧时进行LHS查询，出现在右侧时进行RHS查询。

用一句通俗的话来讲，RHS就是取到它的源值。

注意：“赋值操作的左侧和右侧”，并不意味着只是“=”，实际上赋值操作还有好几种形式。

在作用域中查找变量都是RHS，并且查找的规则是从当前作用域开始找，如果没找到再到父级作用域中找，一层层往外找，如果在全局作用域如果还没找到的话，就会报错了：ReferenceError: 某变量 is not defined

所有的赋值操作中查找变量都是LHS。其中a=4这类赋值操作，也是会从当前作用域中查找，如果没有找到再到外层作用域中找，如果到全局变量啊这个变量，在非严格模式下会创建一个全局变量a。不过，非常不建议这么做，因为轻则污染全局变量，重则造成内存泄漏（比如：a = 一个非常大的数组，a在全局变量中，一直用有引用，程序不会自动将其销毁）。

词法作用域

在上面的作用域介绍中，我们将作用域定义为一套规则，这套规则来管理浏览器引擎如何在当前作用域以及嵌套的作用域中根据变量（标识符）进行变量查找。

我们在前面有抛出一个概念：“词法作用域是作用域的一种工作模型”，作用域有两种工作模型，在JavaScript中的词法作用域是比较主流的一种，另一种动态作用域（比较少的语言在用）。

所谓的词法作用域就是在你写代码时将变量和块作用域写在哪里来决定，也就是词法作用域是静态的作用域，在你书写代码时就确定了。

正常来说，当fn1函数执行完毕之后，其作用域是会被销毁的，然后垃圾回收器会释放那段内存空间。而闭包却很神奇的将fn1的作用域存活了下来，fn2依然持有该作用域的引用，这个引用就是闭包。

总结：某个函数在定义时的词法作用域之外的地方被调用，闭包可以使该函数极限访问定义时的词法作用域。

无边泳池，270度的落地窗可以观看无敌江景，也能俯瞰半个长沙city，窗外就是IFS，楼下就是海信广场，下楼就可以去文和友打卡

预订：美团，搜索 一隅泳池设计师民宿 （其他房型没住过，日常差旅来说多少有点小贵了）

四、本地人建议-干货

1、疫情期间,出行前记得“湖南省居民健康卡”，领取健康码

2、出行首选地铁，下载“长沙地铁”，提前充值 20元左右

3、长沙天气变化大，一定要提前查好天气预报。

4、长沙共享电车有很多但是要带好头盔，不可以逆行不能载人

5、住宿首选五一广场，不要为了省几块钱去折腾自己，提前预约好酒店(不是很推荐住民宿，都是照骗)，晚上还可以去逛步行街。

6、想看夜景可以选择在湘江边上，白天晚上，景色都不错，湘江边有两家民宿还是可以，价格不贵

7、长沙很好吃，但好玩的地方不多。所以基本上三天就差不多了！

8、一定要带上学生证岳麓山索道可以半价，橘子洲的小火车也有折扣

9、在市中心玩的时候一定不要打车，五一广场会非常堵，建议坐地铁或者骑车

10、一定要穿上舒服点的平底鞋，去橘子洲和岳麓山都要走很远的路爬山时能坐索道小火车一定要坐，累到自己的话后边的行程也玩不好

11、碰到来搭讪或者来合影的玩偶一定不要理，要么是骗你来消费，要么是索要拍照费

12、湖南省博物馆一定要去，需要提前一周关注“湖南省博物馆”进行预约，记住周一闭馆哦

13、杜甫江阁就在湘江边上，是看烟花的最佳观景台之一，个人觉得没必要上去，因为现在没有烟花

14、李自健和谢子龙这两个景点在一起，但是比较偏没有地铁可以直达，可以坐公交

15、冬瓜山、四方坪夜市都是本地人聚集的夜市，看旅游路线离哪里比较方便，建议去吃一下的

16、长沙的菜真的很辣，不能吃辣的一定要提前和老板说

17、千万别买路边的水果捞，按克收费的，随便一称就是上百

18、长沙景点还是比较集中的,要提前做好规划不然来回跑太赶
