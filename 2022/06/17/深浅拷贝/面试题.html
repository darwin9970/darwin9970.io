1.盒模型

1.1.一个盒子有四个部分组成：content,padding,border,margin

  • content：实际内容
  • border：边框
  • padding：内边距
  • margin：外边距

1.2.盒子模型

标准盒子模型：

width/height + padding + border + margin

IE怪异盒子模型：

width/height + margin

1.3.box-sizing

box-sizing: content-box | border-box | inherit

2.CSS选择器？优先级？哪些属性可以继承？

2.1.选择器

2.1.1.常用的选择器
id选择器 (#box)
类选择器(.box)
标签选择器(div)
后代选择器(.box div)
子代选择器(.box>div)
相邻同胞选择器(.one+.two)
并集选择器(.one,.two)
交集选择器(.one..two)
2.1.2.伪类选择器
:link : 选择未被访问的链接
:visited : 选择已被访问的链接
:active : 选择活动链接
:hover : 鼠标指针浮动在上面的元素
:first-of-type : 父元素的首个元素
:last-of-type : 父元素的最后一个元素
:only-of-type : 父元素的特定类型的唯一子元素
:only-child : 父元素中唯一子元素
:nth-child(n) : 选取父元素中第N个子元素
:nth-last-of-type(n) :选择父元素中第N个子元素，从后往前
:last-child : 父元素中的最后一个子元素
:root : 设置HTML文档
:empty : 指定空的元素
:disable : 选择被禁用的元素
:checked : 选择选中的元素
:not(selector) : 选择非<selector>元素的所有元素
2.1.3.伪元素选择器
::before : 选择器在被选元素前面插入内容
::after : 选择器在被选元素后面插入内容
:first-letter : 用于选取指定选择器的首字母
:first-line : 选取指定选择器的首行
2.1.4.属性选择器
[attribute*=value]: : 选择attribute属性包含value的所有元素
[atrribute^=value]: : 选择attribute属性开头为value的所有元素
[atrribute$=value]: : 选择attribute属性结尾为value的所有元素

2.2.优先级

!important > 内联 > ID选择器 > 类选择器 > 便签选择器 > *

2.3.继承属性

  • 字体系列属性
font: 组合字体
font-family: 规定字体
font-weight: 设置字体粗细 400标准 700加粗
font-size: 设置字体大小
font-style: 定义字体的风格
font-variant: 偏大或偏小的字体
  • 文本系列属性
text-indent: 文本缩进
text-align: 文本水平对齐
line-height: 行高
word-spcing: 单词间距
letter-spacing: 字符间距
text-transform: 文本大小写
direction: 书写方向
color: 文本颜色
  • 元素可见性
visibility
  • 表格布局属性
caption-side: 定位表格标题位置
border-collapse: 合并表格边框
border-spacing: 设置单元格间距
empty-cells: 单元格的边框出现和消失
table-layout: 表格的宽度由什么决定
  • 列表属性
list-style-type: 文字前小点的样式
list-style-position: 小点的位置
list-style: 以上属性的集合
  • 引用
quotes: 设置嵌套引用的引号类型
  • 光标属性
cursor: 光标变成需要的形状

2.4.无继承的属性

  • display
  • 文本属性：vertical-align, text-decoration
  • 盒子模型的属性：width, height, padding, border, margin
  • 背景属性：background-image, background-color, background-position
  • 定位属性：float, position
  • 生成内容属性：content, counter-reset, counter-increment
  • 轮廓样式属性：outline-style, outline-width, outline-color, outline
  • 页面样式属性：size, page-break-before, page-break-after
